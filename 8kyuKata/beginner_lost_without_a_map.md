__Beginner - Lost Without a Map__
### DESCRIPTION:
Given an array of integers, return a new array with each value doubled.

For example:

`[1, 2, 3] --> [2, 4, 6]`
___
#### Solution

```Python
def maps(a):
    return [i*2 for i in a]
```
