__Reversed Strings__
### DESCRIPTION:
Complete the solution so that it reverses the string passed into it.

```
'world'  =>  'dlrow'
'word'   =>  'drow'
```
___
#### Solution

```Python
def solution(string):
    return string[::-1]
```
